\chapter{AI Use}

\textbf{ChatGPT 5.4} was used throughout the work for three main purposes:

\begin{enumerate}

\item \textbf{Generating \texttt{matplotlib} code} by a prompt like ``I need \texttt{matplot} code which generates a retrieval rate plot for each model/tokenizer in a parquet file that has the following columns: ...''

\item \textbf{Improving logical consistency in writing} by pasting paragraphs and prompting ``Read (this|these) paragraph(s)? and give feedback on consistency and if the flow is good.''

\item \textbf{\LaTeX{} formatting help,} especially when trying figure layouts; e.g.\ ``I have these three figures, but I need them in a subfigure with a common caption and floating position spanning a page.''

\end{enumerate}

\noindent
It was also used for smaller one-off coding questions, e.g.\ ``How do I drop duplicates in pandas?'', ``How do I export a pandas dataframe to parquet without index?'' etc.

No content was used without verification or adaptation.
