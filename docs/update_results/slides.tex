\begin{frame}{Detokenization in LLMs}

Tokenization can split up words non-coherently;\\
e.g.\ \texttt{['H',~'eming',~'way']} \\
How do LLMs reconstruct such larger fragments?

\vfill

\begin{itemize}
\item[\textbf{RQ1}] Do a set of attention heads (5\%) drive whole-word representations in LLMs?
\item[\textbf{RQ2}] How does tokenizer choice affect these representations and robustness to zeroing attention head outputs?
\item[\textbf{RQ3}] How does model size affect these representations and robustness to zeroing attention head outputs?
\end{itemize}

\end{frame}

\begin{frame}{RQ1: Targeted vs.\ Random Ablation}
\begin{itemize}
    \item Targeted ablation suffers large performance drop compared to baseline across all tested configurations (except 1)
    \item Random ablation has minor effect in general (except 1)
\end{itemize}

\vfill
\textbf{For example}: in \texttt{qwen2.5-7B}/English, final cumulative word retrieval falls from 81\% to 1\% under targeted ablation, versus falling to 80\% with random ablation.
\end{frame}

\begin{frame}{Takeaways}
\begin{itemize}
\item Specific attention heads seem drive multi-token word representations.
\item Zeroing these heads \textbf{hurts retrieval} (random not so much)
\item Results suggest:
    \begin{enumerate}
        \item Larger models show \textit{specialized attention}.
        \item The tokenizer impacts learning efficiency for representations.
    \end{enumerate}
\end{itemize}
